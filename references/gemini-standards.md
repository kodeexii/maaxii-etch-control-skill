# MaaXII Design Standards (BEM + ACSS)

- **CSS Method**: Class-first, BEM (Block-Element-Modifier).
- **Framework CSS**: Automatic CSS (ACSS).
- **Typography**: Always use responsive ACSS variables (e.g., `var(--text-h1)`).
- **Spacing**: Use ACSS spacing scale (e.g., `var(--space-l)`).
- **Clean Code**: No unnecessary wrappers. Match Etch's lean output.
